Apache Druid
-------------

To connect to Druid setup the URI as in this example : 

```
druid://<User>:<password>@<Host>:<Port-default-9088>/druid/v2/sql
```

Add certificates to the **Root Certificate** field, as follows:

<img src="/static/images/root-cert-example.png" /> 


If firewalls are implemented at the database level, then open the following Preset IP addresses to the database port:

* 35.161.45.11
* 52.32.136.34
* 54.244.23.85
