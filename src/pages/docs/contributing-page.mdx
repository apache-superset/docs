Contributing to Superset
-------------

Superset is currently incubating at the Apache Software foundation. The core contributor team (also known as committers) use [this mailing list](https://lists.apache.org/list.html?dev@superset.apache.org) and [Github issues](https://github.com/apache/incubator-superset/issues) to collaborate.

You can find the most upto date instructions on how to contribute in the Community Contribution Guide
[described in CONTRIBUTING.MD](https://github.com/apache/incubator-superset/blob/master/CONTRIBUTING.md).

Here are some highlights from CONTRIBUTING.MD:

- Read about the [types of contributions](https://github.com/apache/incubator-superset/blob/master/CONTRIBUTING.md#types-of-contributions) you can make
- Learn the protocol for [submitting PR's](https://github.com/apache/incubator-superset/blob/master/CONTRIBUTING.md#protocol)
- Learn how to [setup your local environment for development](https://github.com/apache/incubator-superset/blob/master/CONTRIBUTING.md#setup-local-environment-for-development)
